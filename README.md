# 🎬 HPC Movie Recommendation System

## 📌 Overview
This project implements a movie recommendation system using High Performance Computing (HPC) techniques. It demonstrates how parallel processing improves performance in similarity computation.

## 🚀 Features
- Movie recommendation based on user ratings
- Parallel processing using multiprocessing
- Sequential vs Parallel performance comparison
- Interactive UI using Streamlit
- Visualization of execution time

## 🧠 HPC Concepts Used
- Data Parallelism
- Multiprocessing (shared memory)
- Load Balancing
- Scalability
- Performance Optimization

## 🛠️ Tech Stack
- Python
- Pandas, NumPy
- Streamlit
- Matplotlib

## 📊 Dataset
MovieLens Dataset
